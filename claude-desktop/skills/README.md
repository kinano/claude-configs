# Claude Desktop Skills

These `.skill` files are for **Claude Desktop** and must be installed manually — they don't get picked up automatically.

## How to Install

1. Open **Claude Desktop**
2. Go to **Settings → Skills**
3. Click **Import Skill** (or drag and drop the `.skill` file)
4. The skill will then be available in your Claude Desktop sessions

## Skills

| File | Description |
|------|-------------|
| `farty-bobo-mode.skill` | Activates Farty Bobo persona and Lewis Black voice |
